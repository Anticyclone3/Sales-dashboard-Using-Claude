# 📊 SuperStore Executive Intelligence Dashboard

> An interactive, AI-narrated business intelligence dashboard built for executive decision-making — turning 9,994 raw retail transactions into a boardroom-ready analytical tool with conversational AI.

## 🧭 Overview

This project analyzes 4 years (2018–2021) of US retail transaction data — 9,994 orders across 3 product categories, 4 regions, and 3 customer segments — and turns it into a single-file, fully interactive dashboard designed for a CEO/CFO audience.

The brief I set for myself was simple: **build something a real executive could open cold and understand in under 60 seconds, then drill into for the next 10 minutes.** That meant prioritizing clarity over chart-count, and giving the dashboard a way to *talk back* when a stakeholder has a follow-up question instead of forcing them to interpret a chart themselves.

---

## ❓ The Business Questions This Answers

Before building a single chart, I defined the questions a CFO would actually ask:

1. Where is revenue concentrated, and is that concentration healthy or risky?
2. Which product lines are *profitable* vs. just *high-revenue*? (These are not the same thing.)
3. At what discount level does a "sale" stop being profitable?
4. Which regions/segments are underperforming, and is it a volume problem or a margin problem?
5. Are there seasonal patterns worth planning inventory or staffing around?

Every chart in this dashboard exists to answer one of these — not because it looked good in a gallery.

## ✨ Key Features

### 📈 11 Interactive Visualizations
- Monthly Sales & Profit trend (dual-axis)
- Category revenue breakdown (donut)
- Profit margin by sub-category — color-coded red/amber/green by profitability threshold
- Regional performance (bar + margin% line overlay)
- Customer segment revenue growth by year (stacked)
- Sales-vs-Profit-vs-Quantity bubble map by sub-category
- **Discount impact analysis** — pinpoints the exact discount % where average profit turns negative
- Top 10 states by revenue with margin labels
- Shipping mode distribution + logistics cost table
- Year-over-year growth with auto-calculated % labels
- Monthly order volume heatmap

### 🎛️ Global Filter System
Year, Region, Segment, Category, and Ship Mode filters cascade across **all 11 charts and the KPI bar simultaneously** — built with vanilla JS state management, no framework.

### 🤖 AI Q&A Voice Analyst
The standout feature: a conversational panel where a stakeholder can **type any question** ("Which region should we invest more in?", "What's driving losses?") and get:
- A live answer generated by Claude, grounded *only* in the currently filtered dataset (not the full dataset — so the answer changes if you've filtered to "East + 2019")
- The answer **read aloud automatically** with word-by-word highlighting
- Multi-turn memory, so follow-up questions like "why?" work in context
- Playback controls: pause mid-sentence and **resume from the exact word**, plus adjustable speech speed

### 📱 Responsive & Production-Polish Details
- Breakpoints for tablet and mobile (voice panel becomes a full-screen sheet on small viewports)
- Skeleton loading screens on initial load instead of a blank flash
- Tooltips on every data point, smooth 200ms filter transitions

## 🛠️ Tech Stack

| Layer | Choice | Why |
|---|---|---|
| Charting | Chart.js | Lightweight, no build step, good tooltip/animation defaults |
| Data | Vanilla JS, embedded JSON | Single-file deliverable — no backend, no database needed for this scope |
| AI Layer | Claude (Anthropic API) | Powers the Q&A analyst — system prompt constrains it to answer only from the live filtered data context |
| Voice | Web Speech API (`SpeechSynthesisUtterance`) | Native browser TTS, no external service dependency |
| Styling | Hand-written CSS, custom design system | No Tailwind/Bootstrap — full control over the executive aesthetic (navy/teal palette) |

**No frameworks. No build pipeline. One HTML file.** Open it in any browser and it works.

## 🔍 What I Found in the Data (Sample Insights)

- Technology consistently drives the strongest margins; Furniture is structurally the weakest, with several sub-categories running negative profit
- **Discounts above ~30% are the inflection point** where average profit per order turns negative — a concrete, actionable pricing signal
- Revenue concentration by region doesn't match margin performance — the highest-revenue region is not the highest-margin region, which is the kind of gap a CFO needs surfaced, not buried in a spreadsheet

## 🚀 How to Run

No installation needed.

```bash
# Just open the file in any modern browser
open SuperStore_Executive_Dashboard.html
```

For the AI Q&A feature to work outside of a sandboxed environment, you'd point the fetch call at your own Anthropic API key/proxy — the dashboard itself (charts, filters, KPIs) works fully offline with zero dependencies beyond the Chart.js CDN.
